Pre-requisites:
- Windows 98 or later
- have installed Framework .Net 1.1 runtime

Build instructions:
- run make.bat

Outputs:
- mathgl.dll     optimized library
- mathgld.dll   debug library

Build options:
- define DEBUGONLY to build only the debug library
- define OPTIMIZEDONLY to build only the optimized library

Testing:
- run runtests
- check the results look reasonable
